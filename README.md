### Stack Size Tweaks
Stack Size Tweaks increases the default stack size of items for a better survival experience.

It let's you change stack sizes up to 2.1 billion using the config.

### Default Stack Sizes
Categories are ingredients, blocks, food items, items that normally stack to 16, stews, potions, filled buckets, enchanted books and discs

![Default Stack Sizes](https://cdn.modrinth.com/data/cached_images/f8b0e6b5798cfee704c3da2dac08db81fb9f7ded.png)

### Font
Stack Size Tweaks includes a custom font that only renders inside the inventory (enabled by default).

It makes higher values inside the inventory much more readable alongside the formatting.

![Example image of custom font vs vanilla font](https://cdn.modrinth.com/data/cached_images/3721aa3d3402a3a71c0275ac7bef87a433a20268.png)


### Config
This mod has a config where you can adjust the stack sizes for different item categories and specific items.

Editing max stack sizes requires a restart to take effect.

Access the config by using Mod Menu or NeoForge mods sreen, editing the stacksizetweaks.json5 file in the config folder.
